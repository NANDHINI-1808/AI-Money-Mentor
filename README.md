## 🌟 Overview  
AI Money Mentor is an intelligent web application designed to help users manage their finances smartly using AI-driven insights.  
It simplifies budgeting, investment planning, and financial decision-making.

---

## ✨ Features  

🔹 📊 **Dashboard**  
Track balance, income, and expenses in one place  

🔹 💬 **AI Chat Assistant**  
Get instant financial advice and smart suggestions  

🔹 📈 **Investment Planner**  
Personalized SIP and portfolio recommendations  

🔹 ❤️ **Financial Health Score**  
Analyze your financial stability and improvement areas  

🔹 🔐 **Login System**  
Simple and clean user authentication UI
