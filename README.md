# 💰 AI Money Mentor

AI Money Mentor is a smart financial assistant web application designed to help users manage their money efficiently using AI-powered insights.

## 🚀 Features
- 📊 Dashboard to track balance, income, and expenses
- 💬 AI Chat Assistant for financial advice
- 📈 Investment Planner with SIP recommendations
- ❤️ Financial Health Score analysis
- 🔐 Simple login system

## 🛠️ Technologies Used
- HTML
- CSS (Tailwind + Custom Design)
- JavaScript

## 💡 How It Works
The system analyzes user financial data and provides intelligent suggestions like saving tips, investment plans, and budgeting strategies.

## 🎯 Objective
To simplify personal finance management using AI and help users make better financial decisions.
